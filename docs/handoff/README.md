# 大模型接手文档说明

这个目录保存的是公开脱敏版接手资料，适合发给后续大模型或开发者继续优化。

包含：

- `HANDOFF.md`：完整接手文档，包含项目结构、部署方式、验证方式、已知边界和后续方向。
- `HTML_PRO_DASHBOARD_SUMMARY.md`：html-pro-card 版本总结。
- `CUSTOM_CARD_DASHBOARD_SUMMARY.md`：custom-card 版本总结，用于理解两个方案的差异。
- `NEXT_OPTIMIZATION_PLAN.md`：后续功能和交互优化清单。

公开版已经替换本地 HA 地址为 `<HA_HOST>`，没有包含 SSH 密码、HA 长期访问令牌或其他私密凭据。
